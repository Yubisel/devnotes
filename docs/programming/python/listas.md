# Listas

## Creación

```python
lista = [1, 2, 3]
```

## Copiar o duplicar una lista (copy)

```python
lista2 = lista.copy()
```

## Agregar (append)

```python
lista.append(4)
```

## Vaciar elementos de una lista (clear)

```python
lista.clear()
```

## Contar ocurrencias de un elemento (count)

```python
lista.count(2)
```

## Contar elementos (len)

```python
len(lista)
```

## Accediendo a elemento por posición

```python
lista[2]  # accede al tercer elemento de la lista
```

## Eliminar elemento devolviendolo a una variable (pop)

```python
ultimo = lista.pop()
```

## Eliminar elemento que no sea el ultimo (remove)

```python
lista.remove(2)
```

## Ordenar (sort)

Para poder ordenar con el método sort todos los elementos de la lista tienen que ser del mismo tipo

```python
lista.sort() 
```

## Revertir elementos (reverse)

```python
lista.reverse()
```
